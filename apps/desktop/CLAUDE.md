# Desktop (Electron + Vite)

This package is the **Electron Forge + Vite** desktop shell for Ordo. Use **Bun** at the repo root (`bun install`, `bun run dev`).

Dev and build are driven by **Electron Forge** and the **Vite plugin** ([docs](https://www.electronforge.io/config/plugins/vite)); do not replace the renderer with `Bun.serve()` here.
